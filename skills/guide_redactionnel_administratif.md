# Guide rédactionnel administratif

Conventions et gabarits pour produire des documents administratifs conformes aux usages de la fonction publique française.

---

## 1. Principes fondamentaux

- **Clarté** : une idée par paragraphe, vocabulaire précis et non ambigu.
- **Concision** : supprimer tout ce qui n'apporte pas d'information utile.
- **Hiérarchie** : aller du général au particulier, du principal à l'accessoire.
- **Neutralité** : ton impersonnel, pas de jugement de valeur ni d'écriture inclusive.
- **Traçabilité** : toujours référencer les textes fondateurs (lois, décrets, circulaires) avec leur date et numéro exact.

---

## 2. Le courrier administratif

### Structure obligatoire

```
[En-tête de l'administration]               [Lieu, date]
[Objet : ...]
[Référence : ... (si réponse)]
[Pièce(s) jointe(s) : ... (si applicable)]

[Formule d'appel]

[Corps : 3 parties]
  1. Rappel du contexte / objet de la demande (1 §)
  2. Développement / analyse / décision (1-3 §)
  3. Suite donnée / délai / contact (1 §)

[Formule de politesse]
[Signature]
```

### Formules d'appel selon le destinataire

| Destinataire | Formule |
|---|---|
| Particulier | Madame, Monsieur, |
| Agent public | Madame, Monsieur, |
| Élu | Madame la Maire, Monsieur le Président, |
| Magistrat | Madame la Présidente, Monsieur le Procureur, |

### Formules de politesse selon le sens hiérarchique

| Sens | Formule |
|---|---|
| Supérieur → subordonné | « Veuillez agréer, ... l'assurance de ma considération. » |
| Subordonné → supérieur | « Je vous prie d'agréer, ... l'expression de mon profond respect. » |
| Administration → particulier | « Veuillez agréer, ... l'expression de nos salutations distinguées. » |
| Pairs | « Veuillez agréer, ... l'assurance de ma considération distinguée. » |

## 3. La note administrative

### Note d'information
```
NOTE
À    : [destinataire(s)]
De   : [émetteur]
Date : [date]
Objet: [objet précis]

[Contexte en 2-3 lignes]

[Développement structuré avec titres si > 1 page]

[Conclusion / suite attendue]
```

### Note de synthèse
Structure en 4 parties : **Introduction** (problématique) → **Développement** (analyse des éléments) → **Évaluation** (mise en perspective) → **Conclusion** (réponse à la problématique).

## 4. Le compte rendu de réunion

```
COMPTE RENDU DE RÉUNION

Réunion : [intitulé]
Date     : [date]
Lieu     : [lieu]
Président: [nom, qualité]
Présents : [liste]
Excusés  : [liste]
Absents  : [liste]

ORDRE DU JOUR
1. ...
2. ...

DÉROULEMENT

1. [Premier point]
   [Résumé des échanges en style indirect. Interventions attribuées.]
   → Décision : ...
   → Action : [responsable] pour le [date]

2. [Deuxième point]
   ...

PROCHAINE RÉUNION : [date, lieu]

Le [date],
[Signature du rédacteur]
```

## 5. Le rapport

### Structure type

1. **Page de garde** : titre, auteur, destinataire, date, mention de confidentialité si nécessaire.
2. **Résumé exécutif** (1 page max) : problème, méthode, conclusions principales, recommandations.
3. **Sommaire** avec numéros de page.
4. **Corps du rapport** :
   - Introduction (contexte, périmètre, méthode)
   - Développement (parties numérotées 1., 1.1., 1.1.1.)
   - Conclusion et recommandations
5. **Annexes** numérotées et référencées dans le corps.

### Numérotation des parties
```
1. Titre de partie
   1.1. Sous-partie
        1.1.1. Sous-sous-partie
```

## 6. L'arrêté

Structure obligatoire d'un arrêté :

```
[Timbre de l'administration]

ARRÊTÉ [objet]

[Le ministre / Le directeur / Le préfet] de [...]

VU [texte fondateur 1] ;
VU [texte fondateur 2] ;
VU [avis / délibération si applicable] ;

CONSIDÉRANT [motif principal] ;
CONSIDÉRANT [motif secondaire si nécessaire] ;

ARRÊTE :

Article 1er. – [Disposition principale]

Article 2. – [Dispositions complémentaires]

Article 3. – [Entrée en vigueur / publication]

Article 4. – [Exécution : désignation des responsables]

Fait à [lieu], le [date]

[Signature]
```

**Règles spécifiques :**
- Chaque visa commence par « VU » en majuscules, se termine par ` ;`
- Les considérants commencent par « CONSIDÉRANT » en majuscules
- Le dispositif commence après « ARRÊTE : »
- Le dernier article est toujours l'article d'exécution

## 7. La fiche de procédure

```
FICHE DE PROCÉDURE

Titre         : [intitulé de la procédure]
Référence     : [code / numéro interne]
Version       : [numéro]
Date          : [date de création ou de dernière révision]
Rédacteur     : [nom, service]
Approbateur   : [nom, qualité]
Destinataires : [services concernés]

OBJET
[Description en 2-3 lignes de ce que fait cette procédure]

DOMAINE D'APPLICATION
[Qui est concerné, dans quelles circonstances]

DOCUMENTS DE RÉFÉRENCE
- [Texte 1]
- [Texte 2]

DESCRIPTION DE LA PROCÉDURE

Étape 1 — [Intitulé]
  Acteur    : [qui]
  Action    : [quoi]
  Délai     : [quand]
  Document  : [pièce produite ou utilisée]

Étape 2 — [Intitulé]
  ...

INDICATEURS DE SUIVI
- [Indicateur 1 : mesure / fréquence]
- [Indicateur 2]

HISTORIQUE DES RÉVISIONS
| Version | Date | Modifications |
|---------|------|---------------|
| 1.0     | ...  | Création      |
```

## 8. Références aux textes juridiques

Toujours citer dans cet ordre : nature du texte, numéro, date, intitulé, source.

```
Exemples corrects :
- La loi n° 83-634 du 13 juillet 1983 portant droits et obligations des fonctionnaires
- Le décret n° 2016-151 du 11 février 2016 relatif aux conditions et modalités de mise en œuvre du télétravail
- La circulaire du 24 juillet 2018 relative à la simplification des relations entre l'administration et les citoyens
```

## 9. Génération de documents avec export_docx / export_pdf

Pour produire un document Word ou PDF :

- Utiliser `export_docx` pour un document Word éditable.
- Utiliser `export_pdf` pour un document non modifiable (diffusion externe).
- **Rédiger un contenu COMPLET** : utiliser le champ `paragraphs` (liste) pour plusieurs paragraphes par section.
- **Viser au minimum 15 sections** pour un rapport ou une note de fond.
- Ne pas se limiter à un squelette de puces : rédiger de vrais paragraphes développés.
- Un courrier ou un compte rendu court peut avoir 3-5 sections seulement.

```json
{
  "title": "Note relative à...",
  "sections": [
    {
      "heading": "Objet",
      "paragraphs": [
        "La présente note a pour objet de...",
        "Elle fait suite à..."
      ]
    },
    {
      "heading": "Contexte",
      "paragraphs": [
        "Par décision du ..., il a été décidé de...",
        "Cette mesure s'inscrit dans le cadre de..."
      ]
    }
  ]
}
```

## 11. Erreurs à éviter

- ❌ Écriture inclusive (« les agent·e·s ») → ✅ « les agents »
- ❌ Anglicismes non consacrés → ✅ termes français officiels
- ❌ Phrases de plus de 3 lignes
- ❌ Références vagues (« la loi prévoit... ») → ✅ toujours citer le texte précis
- ❌ Premier paragraphe sans mise en contexte
- ❌ Conclusion sans suite concrète (décision, action, délai)
- ❌ Arrêté sans article d'exécution final
- ❌ Rapport sans résumé exécutif
